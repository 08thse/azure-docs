# Azure Event Hubs quotas and limits
Event Hubs linked service supports end-to-end event streaming experience in Synapse Studio. You can stream data from Event Hubs by connecting an event hub as a linked service to the Synapse workspace and then process the event data using downstream applications such as Data Explorer.
