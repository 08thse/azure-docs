# Service request failed. Status: 404 (Not found)

This article describes causes and resolution steps for receiving a 404 error from service requests to Azure Digital Twins. 

## Symptoms

This error may occur on many types of service requests that require authentication. The effect is that the API request fails, returning an error status of `404 (Not found)`.
