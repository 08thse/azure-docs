This Azure Communication Services Call Automation AI sample demonstrates how the Call Automation SDK can be used with Azure Cognitive Services and Open AI to help improve customer experience.

In this sample, you will learn how to answer a call, play a prompt using text to speech, recognize user voice input, send that input through to Open AI for it to answer, play Open AI generated answers back to callers and end how to end calls.

If you would like to download this project and test this your self you can find it [here](aka.ms/ca-ai-sample).
