# Azure IoT Hub extension for Visual Studio Code

Visual Studio Code (VS Code) lets you add languages, debuggers, and tools to your installation to support your development workflow. The Azure IoT Hub extension for Visual Studio Code lets you add Azure IoT Hub functionality to your VS Code installation, so you can manage, develop, and interact with your Azure IoT Hub hubs, devices, and modules during development.

# Installation

You can install the extension from the Visual Studio Code Marketplace, by finding and installing the extension from within Visual Studio Code or [directly from the VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=vsciot-vscode.azure-iot-toolkit). To find and install the extension from within VS Code, perform the following steps.

1. In VS Code, select the Extensions view by either selecting the Extensions view icon from the view bar or by selecting Ctrl+Shift+X.

    :::image type="content" source="media/reference-iot-hub-extension/iot-hub-vscode-extensions-view.png" alt-text="Screenshot showing the Extensions view icon and shortcut from Visual Studio Code.":::

1. Enter *Azure IoT Hub* in the search box to filter the Marketplace offerings. You should see the Azure IoT Hub extension in the list.
    
    
1. Select the Azure IoT Hub extension to display the Extension details page.

    Check the extension ID in the More Info section of the Extension details page to confirm that it's set to vsciot-vscode.azure-iot-toolkit, the correct extension ID for the Azure IoT Hub extension.
