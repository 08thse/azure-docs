# Jobs in Azure Container Apps (preview)

Azure Container Apps jobs enable you to run containerized workloads that execute for a finite duration and then terminate successfully. You can use jobs to perform tasks such as data processing, machine learning, and more.

## Job types

There are three types of jobs:

- **Manual jobs** are triggered on-demand.
- **Scheduled jobs** are triggered on a schedule and run repeatedly.
- **Event-based jobs** are triggered by a scaling rule.

### Manual jobs

Manual jobs are triggered on-demand. You can trigger a manual job from the Azure CLI or the Azure Container Apps REST API.
