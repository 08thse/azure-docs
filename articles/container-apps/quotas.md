# Quotas for Azure Container Apps

The following quotas exist for Azure Container Apps Preview.

| Feature | Quantity |
|---|---|
| Environments | 2 |
| Container apps per environment | 5 |
| Replicas per container app | 25 |
| Cores per replica | 2 |
| Cores per environment | 50 |
