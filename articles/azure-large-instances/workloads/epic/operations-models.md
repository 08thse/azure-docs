# Operations model
There may be scenarios where customers need to contact Microsoft for assistance. These scenarios may vary but can broadly be grouped into the following categories:  

## Support Incident 
Epic on Azure ALI  is not working as expected from the Azure infrastructure standpoint. Customer opens a support request (SR) to get assistance from Microsoft Azure Support team.   

## Service request 
Customer needs to make changes for Epic on Azure ALI e.g., additional blades/storage. Customer reaches out to Epic on Azure BMI TSP in Microsoft strategic pursuit team (SPT), or CSA in Microsoft cloud solutions architect team (CSA) or Partner enterprise architect team (PEAT).
