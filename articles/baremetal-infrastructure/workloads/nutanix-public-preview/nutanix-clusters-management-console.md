# Nutanix Clusters Management Console

There are three management consoles namely Nutanix Clusters Console, Prism Element Web Console and Prism Central Web Console. 
 
Nutanix Clusters Console: Use Nutanix Clusters Console to Create, hibernate, resume, update, and delete a Nutanix cluster running on Azure. 
 
Prism Element Web Console: Use Prism Element Web Console to manage routine Nutanix tasks in a single console. For example, creating a UVM. 
 
See Prism Web Console Guide for more information on how to manage Nutanix tasks. 

Prism Central Web Console: Used to manage multiple Nutanix clusters. 

See Prism Central Guide for more information on how to manage multiple Nutanix Clusters.
