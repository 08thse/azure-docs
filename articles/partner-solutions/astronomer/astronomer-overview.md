# What is <astronomer> – An Azure Native ISV Service?

Azure Native ISV Services enable you to easily provision, manage, and tightly integrate independent software vendor (ISV) software and services on Azure. This Azure Native ISV Service is developed and managed by Microsoft and F5.

You can find <astronomer> – An Azure Native ISV Service in the [Azure portal](https://portal.azure.com/#view/HubsExtension/BrowseResource/resourceType/astronomer.astronomerPLUS%2FastronomerDeployments) or get it on [Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/f5-networks.f5-astronomer-for-azure?tab=Overview).

In this article you learn how to enable deeper integration of the astronomerSaaS service with Azure.

<astronomer> delivers secure and high performance applications using familiar and trusted load balancing solutions. Use <astronomer> as a reverse proxy within your Azure environment.

The <astronomer> offering in the Azure Marketplace allows you to manage astronomer in the Azure portal as an integrated service. You can implement <astronomer> as a monitoring solution for your cloud workloads through a streamlined workflow.

You can set up the <astronomer> resources through a resource provider named astronomer.astronomerPlus. You can create and manage <astronomer> resources through the Azure portal. astronomer owns and runs the software as a service (SaaS) application including the astronomer accounts created.

Here are the key capabilities provided by the <astronomer> integration:

- **Seamless onboarding** of <astronomer> software as an integrated service on Azure.
- **Unified billing** of <astronomer> through Azure monthly billing.
- **Single-Sign on to <astronomer>** - No separate sign-up needed from astronomer portal.
- **Lift and Shift config files** - Ability to use existing Configuration (.conf) files for <astronomer> deployment.

## Pre-requisites for <astronomer>

### Subscription owner

The <astronomer> integration can only be set up by users who have Owner access on the Azure subscription. Ensure you have the appropriate Owner access before starting to set up this integration.

## Find <astronomer> in the Azure Marketplace

1. Navigate to the Azure Marketplace page.

1. Search for _<astronomer>_ listed.  

1. In the plan overview pane, select the **Subscribe**. The **Create <astronomer>** form opens in the working pane.
