# Troubleshooting Dynatrace integration with Azure

This document contains information about troubleshooting your solutions that use Dynatrace.

## Unable to create an Dynatrace resource

Dynatrace integration with Azure can only be set up by users who have *Owner* access on the Azure subscription. [Confirm that you have the appropriate access](../../role-based-access-control/check-access.md).
