The default audio streaming format is WAV (16kHz or 8kHz, 16-bit, and mono PCM). Outside of WAV / PCM, the compressed input formats listed below are also supported
using GStreamer.

- MP3
- OPUS/OGG
- FLAC
- ALAW in wav container
- MULAW in wav container
