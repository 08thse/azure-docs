The Speech SDK supports Windows 10 and Windows Server 2016, or later versions. It's possible to use parts of the Speech SDK with earlier versions of Windows, although it's not supported or advised.

### System requirements

To use the Speech SDK on Windows, you must install the <a href="https://support.microsoft.com/help/2977003/the-latest-supported-visual-c-downloads" target="_blank">Microsoft Visual C++ Redistributable for Visual Studio 2019 </a>.

- <a href="https://aka.ms/vs/16/release/vc_redist.x86.exe" target="_blank">Install for x86 </a>
- <a href="https://aka.ms/vs/16/release/vc_redist.x64.exe" target="_blank">Install for x64 </a>
- <a href="https://aka.ms/vs/16/release/vc_redist.arm64.exe" target="_blank">Install for ARMx64 </a>
