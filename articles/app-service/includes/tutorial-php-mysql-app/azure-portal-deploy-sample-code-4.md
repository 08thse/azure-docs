In the Deployment Center page:

1. Select **Logs**. A deployment run is already started.

1. In the log item for the deployment run, select **Build/Deploy Logs**.

    You're taken to your GitHub repository and see that the GitHub action is running. If you let it finish, it builds and deploys your code in two separate stages, as defined in the workflow file.

1. Since you'll make some changes to your app, select **Cancel workflow** to save time.
