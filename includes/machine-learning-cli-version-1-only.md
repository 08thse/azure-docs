> [!IMPORTANT]
> The Azure CLI commands in this article require the 1.0 extension for Azure Machine Learning. To find which version of the extension you have, use the `az --version` command. Use the list of __Extensions__ returned to determine the CLI extension version:
>
> * `azure-cli-ml` - Indicates that you have the __1.0_ extension.
> * `ml` - Indicates that you have the __2.0__ extension.
