1. Sign in to the local Web UI on your device. Go to **Connect and copy**. Select **SMB** to get the access credentials for the shares associated with your storage account.<!--Verify step 1 for the local web UI.-->

    ![Screenshot showing the Connect And Copy page in the local Web UI for a Data Box. The Connect And Copy menu item and the SMB option are highlighted.](media/data-box-get-share-credentials/get-share-credentials-01.png)

2. In the **Access share and copy data** dialog box, copy the **Username** and the **Password** corresponding to the share. Then select **OK**.
    
    ![Screenshot showing the Access Share And Copy Data dialog box in the local Web UI for an SMB share on the Data Box. The Copy icon for the Username and Password and the OK button are highlighted.](media/data-box-get-share-credentials/get-share-credentials-02.png)<!--Remove box from the copy icon for Storage Account.-->
