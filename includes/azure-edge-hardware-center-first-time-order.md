If you are placing a first-time order for Azure Stack Edge, you will need to fill out a web form. The web form collects the following details about your deployment: 

- Total deployment scale 
- Cluster size 
- Hardware requirements 
- Geographic presence 

Microsoft will evaluate your deployment details and may contact you by email for more information. If you have questions, you can send email to <mailto:AzureStack1Pinquiry@microsoft.com>.
