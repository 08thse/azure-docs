## Prerequisites

To work through this tutorial, first upload a virtual hard disk (VHD) file to the Azure Storage account for the lab where you want to create the custom image. Select one of the following links to learn how to upload the VHD file:

- [Upload by using the AzCopy command-line utility](../articles/devtest-labs/devtest-lab-upload-vhd-using-azcopy.md)
- [Upload by using the Microsoft Azure Storage Explorer](../articles/devtest-labs/devtest-lab-upload-vhd-using-storage-explorer.md)
- [Upload by using the PowerShell console](../articles/devtest-labs/devtest-lab-upload-vhd-using-powershell.md)
